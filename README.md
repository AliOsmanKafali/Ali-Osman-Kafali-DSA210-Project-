# Ali-Osman-Kafali-DSA210-Project-
Data Science project for DSA210 
## Dataset

This project uses the Spotify Tracks Dataset:
https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset

## Project Goal
The goal of this project is to analyze Spotify tracks data and explore patterns in music features such as popularity, tempo, and energy.

## Tools
- Python
- Pandas
- Matplotlib

## Planned Analysis
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualization of trends

Additional Steps:
- Handling missing values (if present)
- Basic data transformation (normalization if needed)

Insights will be derived from relationships between musical features and popularity.
